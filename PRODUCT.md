# Product

## Register

product

## Users

主要用户是在手机上利用碎片时间游玩的轻度益智游戏玩家。他们希望无需学习复杂系统即可开始，通过观察、等待和一次精准切割获得即时的释放感。游戏也支持在电脑浏览器中使用鼠标试玩，便于开发、演示和嵌入个人网站。

## Product Purpose

“一笔留白”是一款手机竖屏水墨切割小游戏。玩家避开不断移动的墨刃，切除没有墨刃的画卷区域，逐步达到目标留白比例。成功标准是切割准确、碰撞可信、反馈解压，并能以轻量网页形式通过 GitHub Pages 独立访问或嵌入其他网站。

## Brand Personality

精准、锋利、克制。

拖动阶段安静而专注，成功切割瞬间明确、有重量、有释放感。文字简短直接，水墨意境服务于玩法判断，不使用故作高深的古风表达增加理解成本。

## Anti-references

- 不做营销落地页，不在开始游戏前展示大段品牌介绍。
- 不延续旧版横屏画布、面板堆叠和过多常驻按钮。
- 不使用廉价粒子数量掩盖切割规则或碰撞精度问题。
- 不使用大面积玻璃拟态、紫蓝渐变、圆角卡片阵列或装饰性动效。
- 不照搬 iSlash 的名称、素材、地图、界面、音频或代码。
- 不把水墨风等同于泛黄宣纸、低对比文字和模糊背景。

## Design Principles

1. 规则先于装饰：玩家看到的切线、碰撞位置和清除结果必须一致。
2. 安静蓄势，瞬间释放：反馈通过节奏、声音、轻震和面积变化共同完成。
3. 首玩即懂：玩家应在 30 秒内通过互动理解核心规则。
4. 移动优先：竖屏触控是主要体验，电脑鼠标使用同一套规则。
5. 轻量可嵌入：保持快速启动、稳定帧率和纯静态发布能力。

## Accessibility & Inclusion

- 关键状态不能只依赖颜色表达，危险同时使用形态、运动和声音提示。
- 正文和按钮文字至少满足 WCAG AA 对比度要求。
- 所有界面控件支持键盘焦点和清楚的可访问名称。
- 尊重 `prefers-reduced-motion`，减少震屏、粒子和过渡，但不移除必要状态反馈。
- 声音与设备震动均可关闭，关闭后不影响规则理解和完成关卡。
- 触控目标至少 44 x 44 CSS 像素。
