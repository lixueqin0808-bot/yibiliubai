# 一笔留白 MVP

这是一个桌面端网页原型，直接打开 `index.html` 即可试玩。

## 玩法

- 用鼠标按住并拖动，画出一条笔锋。
- 笔锋穿过浓墨后，会切除没有墨灵的那一块。
- 笔锋碰到墨灵或朱印会损失一笔。
- 留白比例达到目标后过关。

## 当前版本内容

- 15 个测试关卡
- 5 组难度阶段：基础教学、面积策略、多个墨灵、朱印障碍、综合挑战
- 水墨宣纸视觉
- 背景山水随留白比例逐渐显影
- 章节提示与章节命名
- 过关朱砂印章动画
- 桌面与移动端基础响应式布局
- 触屏拖拽基础适配
- Web Audio 合成音效与音效开关
- 墨灵移动障碍
- 朱砂印固定障碍
- 笔画限制
- 留白进度条
- 过关、失败、重画本关、续卷
- 详细玩法说明、元素图例和第一关示范线
- 根据失败原因显示不同提示
- 成功切割后的墨雾消散与切线残影
- 拖线靠近障碍时的红色危险预警

## v0.2 优化记录

- 补充首关教程，明确“浓墨、墨灵、朱印、笔锋”的含义。
- 第一关增加虚线示范，让玩家知道应该从哪里拖到哪里。
- 无效切割、碰到墨灵、碰到朱印、两边都有墨灵等情况会给出不同解释。
- 按钮文案调整为“玩法、重画本关、续卷”，更贴合水墨主题。

## v0.3 优化记录

- 成功切割时，被清除区域会短暂淡出并散成墨点。
- 松手后会留下短暂切线残影，让玩家看清这一笔的位置。
- 碰到墨灵或朱印时，危险物会出现红色警示圈和文字提示。
- 拖动过程中，如果切线靠近墨灵或朱印，会提前显示红色预警圈。
- 无效切割也会有短暂红色线条反馈，减少“没有反应”的感觉。

## v0.4 优化记录

- 关卡从 5 关扩展到 15 关。
- 第 1-3 关：基础教学，单墨灵、大块浓墨、目标较低。
- 第 4-6 关：面积策略，目标提高，形状更不规则。
- 第 7-9 关：多个墨灵，开始要求玩家把墨灵留在同一侧。
- 第 10-12 关：朱印障碍，加入固定红色障碍并限制切割路线。
- 第 13-15 关：综合挑战，多个墨灵和多个朱印组合出现。
- 终局文案同步更新为 15 关完成。

## v0.5 优化记录

- 增加 5 个章节信息：初入画境、留白取势、墨池惊灵、朱砂封印、山河显影。
- 每关开始时显示章节提示，帮助玩家理解当前阶段目标。
- 留白比例越高，背景山水、月影和竹影越清楚。
- 过关时先播放朱砂印章“留白已成”动画，再显示续卷弹窗。
- 顶部关卡标题改为“章节名 · 关卡名”，增强连续闯关感。

## v0.6 优化记录

- 移除桌面版 1024px 最小宽度限制，舞台可以按设备屏幕缩放。
- 舞台保持 1180:720 比例，避免不同屏幕下画面被拉伸。
- 移动端压缩顶部状态、提示、教程面板和底部按钮。
- 教程面板在小屏幕上支持滚动，避免遮挡整屏。
- Canvas 增加触屏拖拽适配，阻止手机浏览器默认滚动和长按行为。
- 竖屏手机会提示“横屏体验更好”，为后续手机版布局做准备。

## v0.7 优化记录

- 增加音效开关，玩家可以在底部切换“音效开 / 音效关”。
- 使用 Web Audio 合成音效，不依赖外部音频文件，保持离线可用。
- 起笔时播放轻微笔触声。
- 成功切割时播放墨雾散开和上扬提示音。
- 无效切割、碰到墨灵、碰到朱印时播放短促失败音。
- 过关盖章时播放低频印章声。
- 音频在第一次点击或触屏后解锁，适配浏览器自动播放限制。
