# 一笔留白 v2

手机竖屏水墨切割小游戏。当前版本完成 15 关基础战役，使用同一套多边形切割、墨刃碰撞、三次机会、墨铁封边和进度目标规则。

## 十五张地图

1. 圆角墨板
2. 收束玉板
3. 三角形
4. 盾牌形
5. 六角玉板
6. 菱形风筝
7. 五角星盘
8. 水滴形
9. 叶片形
10. 山峰形
11. H 形
12. 心形
13. 竖向竹简
14. 灯笼形
15. 八卦阵

星盘、叶片、六角玉板、竹简、灯笼和八卦阵使用低对比度内部纹样加强辨识，不参与碰撞或切割判定。

## 本地运行

```powershell
npm.cmd install
npm.cmd run dev
```

## 验证

```powershell
npm.cmd run build
npm.cmd run test
npm.cmd run test:e2e
```

## 后续边界

当前圆弧轮廓使用 10 至 16 段安全多边形近似。真正平滑的曲线渲染、复杂凹形地图、窄缝结构和组合地图留到独立阶段处理，避免破坏现有切割与反弹稳定性。
