# 结算语音素材清单

结算先以静音语音占位运行。正式素材到位后，五个评价必须使用同一个声音人格，且只替换音频映射，不重做结算逻辑。

| 文件名 | 文案 | 建议时长 |
| --- | --- | --- |
| `hangbao-01.wav` | 夯爆了 | 0.9-1.4 秒 |
| `top-01.wav` | 顶尖 | 0.4-0.9 秒 |
| `ren-shang-ren-01.wav` | 人上人 | 0.4-0.9 秒 |
| `npc-01.wav` | NPC | 0.4-0.9 秒 |
| `la-wan-le-01.wav` | 拉完了 | 0.4-0.9 秒 |

交付要求：WAV，44.1kHz 或 48kHz，单声道优先，16-bit 或 24-bit PCM；同一说话人、同一麦克风距离、同一房间声；峰值低于 -1dBFS；没有背景音乐、长混响或削波。文件内只保留台词和自然呼吸，不添加其他角色或音效。
