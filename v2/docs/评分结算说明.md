# 评分与结算

每关达成留白目标后都会结算。挥墨次数不限，也不扣分；评价只看完成用时和剩余墨点。

## 五档评价

1. 夯爆了
2. 顶尖
3. 人上人
4. NPC
5. 拉完了

时间决定基础得分。三枚墨点均在时可以获得最高评价；剩两枚墨点时最高为“顶尖”；剩一枚墨点时最高为“人上人”。三枚墨点耗尽则按失败流程重开，不进入结算。

每关会保存最快完成时间。刷新网页后，已解锁关卡和最佳时间都会保留。结算页中的“挥墨次数”只用于记录这次游玩的过程，不参与评分。

## 当前素材状态

结算视觉已使用金色书法字和朱印动画。五条正式同声人格语音仍为空位，系统会保持静音语音回退；语音素材到位后只替换音频，不改动评分和结算流程。
