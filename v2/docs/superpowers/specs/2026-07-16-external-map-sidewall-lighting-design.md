# 外扩地图侧壁与定向光设计

## 目标

把当前“向地图内侧压入的封边”整体替换为一块具有真实厚度的浓墨画板：原多边形仍是画板的顶面和全部玩法边界，侧壁从顶面边缘向外展开；左上主光照亮左上侧壁，右下方出现完整投影。

## 已确认的几何定义

- 原始地图多边形是顶面轮廓，不能缩小、内凹或改写。
- 每条顶面边是对应侧壁的短边，也是刀锋和切线的真实物理边界。
- 每条侧壁向地图外侧延展固定深度；外侧轮廓的边更长，相邻外侧边在顶点处相接，构成完整的外扩框架。
- 侧壁由“顶面原边 + 固定长度的外扩长边”组成梯形。每个顶点由短 45 度连接面封口；不使用无限延长的斜接，因此锐角和凹角都不会产生长刺、裂缝或翻折。
- 普通侧壁只使用石墨灰切面材质，材质本身保持低对比；代码依据侧壁朝向，把同一张贴图调成左上亮、中间灰、右下暗，以表现定向光造成的厚度。
- 标记为墨铁封边的边，直接以同一块侧壁位置替换为“冷银锻钢护条”；它不是叠加在顶面上的第二条边。护条必须有窄而清晰的银白受光脊、内侧深铁灰凹槽和规整的金属边界，使其在十余像素宽度下也能一眼区别于普通石墨切面。
- 地图被切掉时，保留下来的新多边形重新生成顶面、外扩侧壁与阴影；被切掉的部分不会留下金属条。

## 光照和阴影

- 固定主光从左上方向照向右下。
- 顶面保留现有石墨浓墨纹理，并加极轻的左上亮、右下暗的渐层，不改变材质主体。
- 面向左上方的侧壁更亮；侧向为中灰；右侧和下侧更暗。冷银锻钢护条遵循同一方向光，但保留更硬的银白高光脊和更深的铁灰凹槽，以便和普通切面一眼区分。
- 阴影按外扩后的完整轮廓生成，向右下偏移，带有限模糊和半透明叠影；它表现画板离纸面的高度，不覆盖顶面。

## 玩法与物理边界

- 刀锋运动、切线起止、切割判定和反弹都继续使用原始顶面多边形。
- 外扩侧壁和投影只用于绘制，不能把可玩区域向内挤压，也不能制造新的内嵌障碍。
- 墨铁封边仍禁止从该顶面边发起或穿越切割，并保留即时“叮”与火花反馈；它不再通过地图内部的梯形障碍物实现。
- 这次会删除旧的 `metalObstacles` 路径，避免视觉边和物理边有两套不一致的模型。

## 渲染顺序

1. 绘制外扩轮廓的右下投影。
2. 绘制所有外扩侧壁，墨铁边在对应侧壁位置替换普通侧壁。
3. 绘制原始顶面多边形及其浓墨纹理与轻微主光渐层。
4. 绘制引导线、墨刃、切线和交互反馈。

## 验收标准

- 顶面地图轮廓没有内缩，所有侧壁均在地图外部。
- 每个顶点的外侧边都由短 45 度连接面连续相接，没有白缝、黑色缺口、尖刺或额外圆角压扣。
- 左上侧壁明显更亮，右下侧壁更暗；地图右下有可见但不过重的投影。
- 冷银锻钢护条和普通石墨切面在材质上清楚可辨，但共用同一侧壁几何，因此贴合无缝；护条不再被误读成一条灰色切面。
- 刀锋不会因为视觉外扩而穿出或提前反弹；它仍在顶面边界内反弹。
- 切走含墨铁边的一侧后，该边随地图一起消失。

## 范围外

- 本轮不重做地图图片、不生成新美术、不调整音效。
- 本轮不改变关卡布局、目标比例或墨刃数值。
