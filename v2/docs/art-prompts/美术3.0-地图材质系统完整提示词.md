# 美术 3.0：地图材质系统完整提示词

这是一套从头设计的地图材质系统，不是在旧地图上局部修改。需要依次生成三张素材：浓墨地图面、普通梯形切面、墨铁封边。现有象牙白四锋与五锋墨刃继续使用，不重新生成。

## 一、统一视觉规则

整体气质：压墨漆板。像宣纸纤维、矿物墨与哑光漆层压制成的硬质游戏板，保留水墨气息，但具有清楚、稳定的实体边界。

亮度层级：

1. 象牙白墨刃最亮。
2. 墨铁封边的冷银灰高光次亮。
3. 普通切面使用中灰与深灰。
4. 地图面保持中深冷灰，是安静的背景层。

统一色彩：中深石墨灰、烟灰、炭黑、少量低饱和冷银灰。禁止黄色、黄绿色、金色、铜色、棕色、木色、饱和蓝色和饱和绿色。朱红只用于墨刃中心与游戏反馈，不进入地图和封边材质。

统一质感：哑光、克制、细腻、低反射。不要油腻高光，不要塑料感，不要沥青感，不要水泥墙感，不要脏污霉斑，不要大块重复水渍。

## 二、生成顺序

必须一张一张生成，不要让 AI 在一张图里同时画三种素材。

1. 先生成“浓墨地图面”。
2. 将浓墨地图面作为参考图，生成“普通梯形切面边条”。
3. 将浓墨地图面和普通切面边条同时作为参考图，生成“墨铁封边条”。

最终文件命名：

- `ink-slate-surface-v3.png`
- `ink-slate-bevel-strip-v3.png`
- `ink-iron-edge-strip-v3.png`

## 三、素材 1：浓墨地图面

生成比例：1:1 正方形，建议 2048 × 2048。

参考图：不提供旧地图参考图，从头生成，避免继承旧地图的脏黑和泥泞色彩。

### 直接复制 Prompt

为手机竖屏水墨切割游戏制作一张可无缝平铺的“浓墨地图表面”材质贴图。只生成材质本身，正视图，画面铺满，不出现地图轮廓、边框、物体、文字、角色或场景。

材质概念是“压墨漆板”：由细密宣纸纤维、矿物墨和哑光漆层压制成的硬质游戏板。它不是普通纸张，也不是石头、木板、水泥、沥青或金属。主体颜色为中深冷石墨灰，综合色接近 #404445；较暗区域接近 #2B2E2F；柔和亮部接近 #555A59。整体必须明显比纯黑更轻、更透气，但仍足够深，使象牙白刀锋在上面非常清晰。

表面纹理要安静、均匀、克制。保留细微横向和斜向的干笔纤维、轻薄墨层叠染、非常低对比度的烟灰晕染，以及少量矿物墨颗粒。纹理尺度以中小尺度为主，不能出现大片黑色水渍、明显云团、脏污霉斑、强烈裂纹、刮痕、中心图案或视觉焦点。画面四个方向的亮度和纹理密度保持均衡。

光照为均匀柔和的正面环境光，没有方向性投影，没有四角压暗，没有中央发亮。表面是高级哑光材质，只允许极弱、宽阔、低对比度的柔光变化，不要镜面反光，不要油亮，不要塑料质感。

这张图必须可上下左右无缝平铺，所有边缘的颜色、明暗和纹理自然衔接。生成结果用于代码裁切成各种异形多边形地图，因此不得自带任何边缘、厚度、倒角或阴影。

绝对禁止：黄色、黄绿色、金色、棕色、木纹、竹纹、铜锈、翡翠色、饱和蓝色、纯黑大色块；不要水泥墙、沥青路、皮革、岩石裂缝；不要文字、印章、红色元素、装饰图案、边框和场景。

### 验收标准

1. 第一眼是冷灰水墨硬板，不是黑墙、泥地或旧纸。
2. 象牙白墨刃放在上面能立即识别。
3. 缩小到手机地图尺寸后仍有细腻层次，但不会显脏。
4. 四边可以无缝平铺，没有明显接缝和重复中心。

## 四、素材 2：普通梯形切面边条

生成比例：4:1 横图，建议 2048 × 512。

参考图：上传刚生成的 `ink-slate-surface-v3.png`。不要上传旧边条。

背景：纯洋红色 #FF00FF，无渐变，方便后续自动抠图。

代码用途：AI 只生成一条横向、可平铺的切面材质带。梯形轮廓由代码裁切，地图多边形原边是梯形长边，短边朝地图内部。不要让 AI 直接画梯形。

### 直接复制 Prompt

参考我提供的“浓墨地图表面”图片，为同一块压墨漆板制作它的“普通内凹切面边条”游戏贴图。必须延续参考图的冷灰色温、宣纸纤维、矿物墨颗粒和高级哑光质感，让地图面与切面看起来属于同一块实体材料。

画面只出现一条从最左延伸到最右的水平窄材质带，正视图、完全平面、没有透视。左右两端纹理与亮度必须自然衔接，能够在代码中横向无缝重复平铺。不要画地图整体，不要画梯形，不要画边框拐角。

切面比地图表面略亮、略密实，但不能变成银色金属。切面的主体为冷调中灰石墨色，约接近 #5A5E5D；靠近地图原边的一侧是一条很细、柔和的冷灰高光，表现被压实后的边缘；中部保留简化的细纤维和矿物颗粒；朝地图内部的一侧逐渐过渡为一条窄而柔和的深灰内影，接近 #303334。整体形成“细高光—中灰切面—深灰内影”三层关系，让地图有轻微厚度，但不夸张。

考虑到素材最终在手机上只显示约 10 至 14 像素高，所有纹理必须大而简洁、轮廓清楚，不能使用细碎花纹、密集裂纹或微小装饰。切面材质带的可见高度约占整张画面高度的 20%，放在画面正中央；上下其余区域全部保持纯洋红色 #FF00FF。主体边缘清晰，不要向洋红背景投射阴影。

绝对禁止：黄色、黄绿色、金色、铜色、棕色、木纹、竹节、青苔、铜锈；不要螺丝、铆钉、压扣、文字、花纹、裂缝、徽章；不要镜面金属，不要发光，不要梯形外轮廓，不要圆角胶囊形状。

### 验收标准

1. 与地图面属于同一材料家族，但切面比地图面稍亮。
2. 缩小到 10 像素高仍能辨认三层明暗关系。
3. 不能看起来像金属条、塑料条或木条。
4. 左右无缝衔接，没有独立主体和明显接头。

## 五、素材 3：墨铁封边条

生成比例：4:1 横图，建议 2048 × 512。

参考图：同时上传 `ink-slate-surface-v3.png` 和 `ink-slate-bevel-strip-v3.png`。

背景：纯洋红色 #FF00FF，无渐变，方便后续自动抠图。

代码用途：它与普通切面占据完全相同的梯形区域；出现锁定边时直接替换普通切面。AI 只生成横向可平铺材质，不生成梯形和转角。

### 直接复制 Prompt

参考我提供的“浓墨地图表面”和“普通内凹切面边条”，为同一款手机水墨切割游戏制作“墨铁封边条”贴图。它是安装在压墨漆板边缘的冷灰墨铁保护层，用于明确告诉玩家这条边不可切割。它必须与地图和普通切面属于同一个视觉世界，但材质辨识度更强。

画面只出现一条从最左延伸到最右的水平窄材质带，正视图、完全平面、没有透视。左右两端必须无缝衔接，可供代码横向重复平铺。不要画地图，不要画梯形，不要画转角，不要画完整金属框。

墨铁封边采用低饱和冷银灰与枪灰色。主体约接近 #747A7B，暗部接近 #3B4041，最亮的细边高光不超过 #AEB4B4。高光必须明显弱于象牙白墨刃，避免抢走游戏主体。靠地图原边的一侧有一条清楚但不刺眼的冷银灰硬边；中部为平整、细微锻压感的枪灰墨铁；朝地图内部的一侧有一条窄而深的接触阴影，使封边像真正嵌入地图，而不是浮在上面的贴纸。

材质纹理要简洁、实体、略带手工锻压感，只保留少量横向拉丝和不规则墨色氧化层，但不能生锈、不能脏、不能出现复杂裂纹。考虑到最终仅显示约 10 至 14 像素高，纹理必须使用宽而清楚的明暗块，不要微小细节。

材质带的可见高度约占整张画面高度的 20%，位于画面正中央；上下其余区域全部为纯洋红色 #FF00FF。主体边缘清晰，不向洋红背景投射阴影。

绝对禁止：黄色金属、黄绿色、金色、铜色、青铜、棕色铁锈、蓝色钢材、彩色反光；不要螺丝、铆钉、压扣、花纹、符号、文字、刀刃、武器造型；不要圆角胶囊，不要厚重立体金属块，不要镜面反射，不要发光。

### 验收标准

1. 放在地图边缘时能立即识别为不可切割的硬质封边。
2. 比普通切面更亮、更硬，但明显暗于象牙白墨刃。
3. 与地图接触处有内嵌感，不像悬浮贴纸。
4. 缩小后仍清楚，左右可以无缝平铺。

## 六、三张图放在一起的总体验收

1. 地图面负责安静承托，不抢墨刃。
2. 普通切面负责厚度，不应被误认为障碍物。
3. 墨铁封边负责规则提示，应比普通切面更硬、更亮。
4. 三者色温一致，均为冷灰体系；不能出现地图偏棕、切面偏绿、封边偏蓝的割裂。
5. 象牙白墨刃始终是画面最亮、最容易追踪的动态对象。
6. 不生成独立顶点压扣。转角、梯形和异形适配全部由代码完成。
