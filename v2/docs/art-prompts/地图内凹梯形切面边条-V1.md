# 地图内凹梯形切面边条 V1

用途：作为所有异形浓墨地图的通用边缘切面。代码会将它横向重复平铺，并裁成“长边贴地图原边、短边朝地图内部”的窄梯形；墨铁封边会替换同一位置。

## 生成参数

- 比例：4:1 横图，建议 2048 x 512
- 背景：纯洋红色 `#FF00FF`，无渐变，方便后续自动抠图
- 主体必须位于画面中央横向区域，上下留出大面积纯洋红色空白
- 不要透明棋盘格，不要场景，不要文字

## 直接复制这段提示词

为手机竖屏水墨切割游戏制作一张“浓墨地图内凹切面边条”UI 贴图素材。画面只出现一条横向延展、可无缝平铺的窄切面材质带，正视图、完全平面、没有透视、没有立体物体整体、没有地图轮廓。

材质设定：这是一块深色浓墨石板被切出后的内凹边缘断面，不是木头，不是竹子，不是金属，不是玉石。整体使用冷调深石墨灰、烟黑、极少量低饱和银灰。上缘只有一条非常细、柔和的冷银灰磨损高光；中部是干净的深灰石墨切面，带低对比度、细微水墨纤维和石墨颗粒；下缘压一条很窄的半透明暗影，让地图表面产生轻微厚度。质感克制、干净、偏高级游戏 UI 材质，能和深灰浓墨地图、象牙白墨刃、银灰墨铁封边协调。

构图要求：边条从画面最左延伸到最右，左右两端的纹理、亮度、线条必须自然衔接，适合代码横向重复平铺。边条可见材质高度只占画面高度约 20%，置于画面正中央；上下其余区域保持纯洋红色 #FF00FF，边缘清晰，方便抠图。边条内部不要复杂花纹，不要裂纹集中在某处，不要螺丝、铆钉、压扣、徽章、符号。

绝对禁止：黄色、黄绿色、金色、棕色、木纹、竹节、铜锈、翡翠色、霓虹色；不要地图整体，不要梯形整体示意图，不要边框，不要按钮，不要文字，不要人物，不要武器，不要背景场景，不要阴影投到洋红背景上。

## 验收标准

1. 缩小到 10 像素高时，仍能看出“浅高光、深切面、暗内影”三层关系。
2. 综合色仅限黑、深灰、冷灰、极少量银灰，不出现黄绿或木质感。
3. 左右拼接时没有明显重复接缝或中央主体。
4. 贴图本身不携带梯形轮廓，梯形由游戏代码裁切生成，才能适配任意异形地图。
